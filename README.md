# Imgur-address-catcher

markdown에 그림을 올릴때 imgur에 업로드 하고 주소를 하나씩 복사+붙여넣기 
하는 작업이 지루해서 만든 스크립트.
실행 후, Ctrl+V를 하면 주소가 복사되어 있습니다!

## Requirement
```pip install -r requirements.txt```

## Usage
```python imgac.py 'image-path'```

## Example
```python imgac.py sample.png```
